SECTION 9 // オペレータ HUD // ABOUT

SECTION 9創作的アーティファクト · サイバーパンク HUD · 低リスク

攻殻機動隊、The Matrix、Metal Gear Solid、新世紀エヴァンゲリオンに着想を得た、シングルページのオペレータ・ダッシュボード。プロダクトとしてではなく、個人的なアーティファクトとして作られた。トラッキングなし、アナリティクスなし、バックエンドなし — ただ HTML、CSS、JS、それに少しの CDN フォントだけ。眺めて、歩き回って、イースターエッグを打ち込んで、終わったら閉じてくれればいい。

01 これは何か

オペレータの HUD：存在しない架空のユニットのためのシミュレートされたコマンドインターフェイス。スカッド・ロースタは、作者のマシン上で実際に動いているサービス（Hermes、Batou、Watchdog、MCPs）を、公安9課の人員として再スタイル化したもの。思考戦車たちは実際の Model Context Protocol サーバである。MAGI の投票は象徴的。コーデックの周波数は冗談。

これはラブレターである。とりわけ 公安9課 という美学とドクトリン — ネットワーク上の異常を見張る、専門家による小さなユニットというアイデア — に対して、そして、その視覚文法を我々に与えた 4 作品に対して。

05 イースターエッグ — 動かし方

COMMAND // INPUT ボックス（HUD の右上）にコマンドを入力して Enter を押す。だいたい 70 個ある。ヒントは 6.5 秒ごとにローテーションする。フルリストは [ ? ] HELP で。

最初に試す好きなものたち：ramiel、tabris、dive、foxhound、kojima、seele、kept you waiting（snake でも可）、follow the white rabbit（rabbit でも可）、それに nanomachines。さらにコナミコード（↑↑↓↓←→←→ B A）。

NETWORK TOPOLOGY パネルの 7 つのノードのいずれかをクリックして、資料を開く。5 つのスカッド・カードはクリック可能。6 つのタチコマもクリック可能。右カラムの 5 つのアクション・ボタン（[PING]、[QUERY]、[DEPLOY]、[INCOMING?]、[REFRESH]）はすべて何かをする。左上のオーディオ・トグルは本物の Web Audio。
